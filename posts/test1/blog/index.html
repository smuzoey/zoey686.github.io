Ubuntu搭建blog的一些指令

Posted at — Jul 6, 2019

1.install git and hugo
 sudo apt install git
 sudo apt install hugo

2.git set user’s information
 git config –global user.name “zoey686”
 git confit –global user.email “邮箱”

 git config –list // show user.name and user.email

2.create a site
 mkdir gitFile //create a folder to store the site
 hugo new site mybolg


3.mybolg generated folder

 1.archetypes Header format
 2.content blog’s artical file
 3.data custom template
 4.layoutsa Website template file
 5.static store pictures and other resorces
 6.themes

4.change theme
 cd themes
 git clone https://…..

5.local booting
 hugo server -t ezhil –buildDrafts

6.deployment server
 hugo –theme=ezhil –baseUrl=“https://zoey686.github.io/" –buildDrafts
 cd public
 git init
 git add .
 git commit -m “……”
 git remote add origin https://github.com/zoey686/zoey686.github.io.git

  /*if origin exist
   git remote rm origin //remove origin
  */

 git push / git push -u origin master

结束
